Electronic Structure of Pairing Hamiltonians

Date Mon 02 May 2016

Understanding high-temperature superconductivity (HTSC) has been a high-profile problem in condensed matter physics since the 1980's. One unique feature of a superconducting system, regardless of its superconducting origin, is that the two-electron reduced density matrix (2-RDM) manifests a large eigenvalue proportional to the system size. This makes the 2-RDM a good probe for when a system undergoes a superconducting transition. To try to understand the basic ingredients of a Hamiltonian that exhibits HTSC David Mazziotti and I studied direct calculations of the 2-RDM for various model Hamiltonians that are known to have pair correlations.

The really exciting thing about this work is:

1) The Hamiltonians we select are generated from products of one-body spin-operators. This is exciting because we can directly see which types of two-body interactions are important.

2) The Hamiltonian has only two-body terms which are projectors onto a \(\eta\)-pairing state-space. The basis functions that parameterize the eigenfunctions of the Hamiltonians are all pairing wave functions

\begin{align} \eta_{i}^{\dagger} = \hat{a}_{i\alpha}^{\dagger}\hat{a}_{i\beta}^{\dagger} \end{align}
\begin{align} |\psi \rangle = \sum_{ n_{1} < n_{2} < ... < n_{N/2} } c_{ n_{1} < n_{2} < ... < n_{N/2} } \hat{\eta}_{n_{1}}^{\dagger}\hat{\eta}_{n_{2}}^{\dagger }... \hat{\eta}_{n_{N/2}}^{\dagger}|\rangle \end{align}

This results in a diagonal 1-RDM for all Hamiltonains regardless of interaction parameters. The diagonal entries are proportional to the electron density on my lattice. Because the 1-RDM contains zero information about the electronic structure of the system this class of Hamiltonians is a really nice test for electronic structure methods aimed at capturing strong electron correlation.

Furthermore, because we have access to the 2-RDM we can calculate off-diagonal 2-RDM "quantum correlation" functions corresponding to \(\eta\)-pairing and superconductivity. For example we can plot

\begin{align} \langle \hat{\eta}_{i}^{\dagger}\hat{\eta}_{i+R}\rangle \end{align}

to see the structure of the off-diagonal order.

This figure was taken from our paper ref. The labels A,B,C,D are labels for various Hamiltonian interaction parameters. The details aren't really that important. What is important is that for 'A' we have an \(\eta\)-pairing correlation function that doesn't decay at long range--i.e. \(R = 8\). For the other labeled points there is some degree of decay. The "long-range" correlation frames Yang's idea of off-diagonal long-range order in a density matrix quite well.

For a long time I had difficulty conceptualizing what this means. In fact, Yang means excatly what he says--Ahh the satisfaction of clear scientific writing. The 2-RDM has elements that don't go to zero as you move away from the diagonal. This structure in the density matrix maniefest one-large eigenvalue. Another way to think about the large eigenvalue is to observe there is a periodicity in the matrix corresponding to one momentum--i.e. the off-diagonal density-matrix elements can be described by one linear-momentum. This is why the eigenvalue that corresponds to that eigenfunction becomes large proportional to system size. The underlying wave function reflects an interaction that has very long correlation length and a definite period.

3) Despite the complexity of the Hamiltonian approximate 3-positivity constriants on the 2-RDM are sufficient for accurate energies and correlation functions.

As I mentioned earlier, this Hamiltonian is superb for testing methods designed to treat strong electron correlation. DQG constraints are a work-horse set of constraints used in quantum chemistry. Here we see that even when entanglement is maximal approximate 3-positivity (DQGT) constraints are sufficient for an accurate simulation of the ground state 2-RDM.
