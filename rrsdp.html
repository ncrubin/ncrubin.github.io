RRSDP

Date Wed 03 February 2016

RRSDP is the name of the method that solves a semidefinite program by relaxing it to a non-linear program. The method was originally developed by Burer and Monteiro (link). I have implemented (link) the RRSDP method in python using the scipy L-BFGS minimizer as a tutorial for the RRSDP algorithm below.

Semidefinite programming is the extension of linear programming to matrix variables. The primal semidefinite program is mathematically stated as follows

\begin{align}\label{Primal} \mathrm{min}\;\; \langle C, X \rangle \\ \mathrm{s.t} \;\; \langle A, X \rangle = b \; ; \; X\succeq 0 \end{align}

where \(C\) is in the space of symmetric matrices (\(C \in \mathcal{S}_{n}\)), \(X\) is in the space of positiv e semidefinite matrices (\(X \in \mathcal{S}_{n}^{+}\)), \(\langle \cdot , \cdot \rangle\) is defined as the trace inner product \(\mathrm{Tr}[C^{T},X]\), and \(b\) is a vector in \(\mathcal{R}^{m}\). The conjugate dual of the primal problem can be determined in the normal way by considering the maximization of dual variables with an inner minimization over the primal.

\begin{align} \mathrm{max}\;\; b^{T} y \\ \mathrm{s.t} \;\; S = C - y^{T}A \; ; \; S\succeq 0 \end{align}

RRSDP solves the primal problem by factoring the primal variable ensuring non -negativity.

\begin{align}\label{factored} X = RR^{T} \end{align}

The factorization transforms the semidefinite program to a non-linear program which is solved via an augmented-Lagrangian (AL) method. The augmented-Lagrangian method balances the penalty method with the Lagrange multiplier method to prevent ill conditioning as the penalty parameter increases. Considering the augmented-Lagrangian

\begin{align}\label{augLagrangeRRSDP} L(R, y, \sigma) = \langle C, RR^{T} \rangle - \sum_{y = 1}^{m} y_{i}(\langle A_{i}, RR^{T} \rangle - b_{i }) + \frac{\sigma}{2}\sum_{i=1}^{m}(\langle A_{i}, RR^{T} \rangle - b_{i} )^{2} \end{align}

and its gradient

\begin{align} \nabla_{R}L(R,y,\sigma) =& 2 \overline{S}R \\ \overline{S} =& C - \sum_{i=1}^{m} \left( y_{i} - \sigma( \langle A_{i}, RR^{T} \rangle - b_{i} ) \right) A_{i} \end{align}

the RRSDP method determines an optimal primal dual pair \((R^{*},(y^{*},\sigma^{*}))\) that minimizes the objective \(\langle C, RR^{T}\rangle\) and satisfies \(\langle A_{i}, RR^{T} \rangle = b_{i}\) for all constraints \(\{i\}\). The optimal multiplier (\(y\)) and penalty (\(\sigma\)) is iteratively determed by the sequence \((y^{k},\sigma^{k})\) that converges to the solution \((y^{*}, \sigma^{*})\). The sequence is produced by the following procedure:

  1. Minimize \(L(\cdot, y, \sigma)\) with respect to \(R\)}
  2. \(\mathrm{compute }\;\; v = ||\langle A, RR^{T} \rangle - b||_{\infty}\)
  3. if \(v < \eta v_{k}\), set

    1. \begin{align} y_{i}^{k+1} =& y_{i}^{k} - \sigma_{k}(\langle A_{i}, RR^{T} \rangle - b_{i} )\\ \sigma_{k+1} =& \sigma_{k}\\ v_{k+1} =& v \end{align}

    2. else,

      \begin{align} y_{i}^{k+1} =& y_{i} \\ \sigma_{k+1} =& \gamma \sigma_{k}\\ v_{k+1} =& v_{k} \end{align}


The outlined procedure is repeated until primal and dual feasibility are below a particular threshold (\(\epsilon\)). For the dual multiplier and penalty update steps, the variable \(\eta\) certifies that there has been a sufficient decrease in the primal feasibility while \(\gamma\) increases the penalty parameter in an attempt to bias the AL problem to the solution. Each minimization of the AL is computed using the low-memory BFGS procedure. It is clear that the update for the Lagrange multipliers ensures the next iteration will be dual feasible for the unaugmented Lagrangian.
